# Sensor de Temperaturas Submarino

## Descripción
Aplicación en C# (Windows Forms) que simula un sensor submarino frente a la costa de Guerrero.  
La aplicación genera lecturas de temperatura (en grados centígrados) y permite calcular:

- Temperatura máxima y mínima diaria
- Promedio diario
- Temperatura máxima, mínima y promedio semanal

## Funcionalidades
- Generación automática de datos de temperatura (24 horas x 7 días).
- Visualización en una tabla tipo DataGridView.
- Cálculo de estadísticas diarias y semanales.
- Limpieza de datos y reinicio del formulario.
- Confirmación al salir de la aplicación.

## Tecnologías utilizadas
- Lenguaje: **C#**
- Entorno: **Visual Studio Community**
- Interfaz: **Windows Forms**
